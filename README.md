# CMDS---2025-2027-Supervised-Applied-Project

ONE STEP AUTOFILL FOR PERSONAL DETAILS

Constantin TONU

The rapid digitalization of services has increased the need for secure, privacy-preserving identity data exchange mechanisms. Current identity management systems are often centralized, cloud-dependent, or limited to authentication workflows rather than dynamic data sharing. This report proposes a privacy-first mobile system for storing encrypted personal and business data locally on user devices and enabling selective sharing through QR-code-initiated peer-to-peer communication. The system emphasizes self-sovereign identity principles, local encryption, real-time user consent, and elimination of persistent cloud storage. The report presents the conceptual background, analyzes the current state of the art in digital identity wallets and self-sovereign identity systems, identifies existing limitations, and outlines the architecture and objectives of the proposed system.
